Mailing Lists related to Panini

For Panini Users

Please use the mailing list: paninij-users@lists.sourceforge.net for questions about Panini code and for help on examples, running the compiler and tools.

For Panini Developers

Please use the mailing list: paninij-develop@lists.sourceforge.net for questions concerning Panini's implementation, for reporting bugs in the compiler and tools and for contributing to Panini's source code.

For Other Questions/Concerns

Please e-mail Hridesh Rajan at the e-mail address hridesh@iastate.edu for more information on Panini's design and for other questions and concerns.
