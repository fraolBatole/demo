Coq Formalization of Panini

This URL makes formalization of the Panini program available.

  • syntax.v This file defines the syntax of the core calculus.
  • staticsemantics.v This file defines the static semantics of the core calculus.
  • dynamicsemantics.v This file defines the dynamic semantics of the core calculus.
  • typesoundness.v This file presents the type soundness proof of the calculus, and also present other theorems and proofs.
  • proofs.v This file presents proofs of the theorems.
  • standardauxiliary.v This file defines the auxiliary functions that are standard, and not specific to the static and the dynamic semantics of Panini.
  • auxiliaryfunctions.v This file defines auxiliary functions used in the static and the dynamic semantics.

All files in the formalization are available via this ZIP archive.
