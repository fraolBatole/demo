May 21, 2014: The Panini compiler version 0.9.3 available below provides capsule-oriented programming features. This is a bug-fixing/performance improvement release which also adds support for declaring methods that return final classes. As is usual, bug fixes from previousrelease are also included. Note that you can always write to us at the e-mail address panini@iastate.edu about the language, compiler, and about the programming language manual.

August 14, 2013: The Panini compiler version 0.9.2 available below provides capsule-oriented programming features. This is a feature release and it includes two language features: implicitly parallel foreach expressions and capsule initializers. It also includes a more advanced sequential consistency analysis compared to previous versions. As is usual, bug fixes from previous release are also included. Note that you can always write to us at the e-mail address panini@iastate.edu about the language, compiler, and about the programming language manual.
