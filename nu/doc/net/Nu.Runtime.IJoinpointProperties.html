IJoinpoint Properties

The properties of the IJoinpoint interface are listed below. For a complete list of IJoinpoint interface members, see the IJoinpoint Members topic.

Public Instance Properties

_Kind Returns the kind of the joinpoint.
Args Returns the list of arguments of the join points. Returns an array of length zero if there are no arguments.
Kind Returns the string representation of the join point kind.
Location Returns the Location of the join point. Please see Nu.Runtime.ISourceLocation for more details.
ReturnValue Returns the return value at the join point if it is a valid return value otherwise null.
Signature Returns the signature of the join point. Please see the interface Nu.Runtime.Signature.ISignature for more details.
StaticPart Returns the static representation at the join point. Please see Nu.Runtime.IStaticPart for more details.
Target Returns the target of the join point. Returns null in cases where there is no target.
This Returns the value of "this" at the join point. Returns null in the case of static join points
