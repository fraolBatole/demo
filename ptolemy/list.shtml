The Ptolemy Programming Language

Mailing Lists Related to the Ptolemy Language

Ptolemy Users

Current or potential Ptolemy users may seek clarifications on the language and its infrastructure by posting to ptolemyj-users mailing list

To post to this discussion list, send an email to:

ptolemyj-users@lists.sourceforge.net

General information about this mailing list is available at:

https://lists.sourceforge.net/lists/listinfo/ptolemyj-users

The link above also provides a web-based mechanism to subscribe to this mailing list.

The archived discussion about Ptolemy is available at:

http://sourceforge.net/mailarchive/forum.php?forum_name=ptolemyj-users

Ptolemy Developers

To seek clarifications on the language implementation, to contribute to the development of Ptolemy's infrastructure, or to simply follow the development of the Ptolemy programming language going forward please post to ptolemyj-commit mailing list. Notifications for any changes in the repositories for the Ptolemy project are also sent to this mailing list.

To post to this discussion list, send an email to:

ptolemyj-commit@lists.sourceforge.net

General information about this mailing list is available at:

https://lists.sourceforge.net/lists/listinfo/ptolemyj-commit

The link above also provides a web-based mechanism to subscribe to this mailing list.

An archive containing messages for this mailing list is available at:

http://sourceforge.net/mailarchive/forum.php?forum_name=ptolemyj-commit
