Install Slede


1. Install TinyOS 1.1.0 for Linux (http://www.tinyos.net/download.html)

2. Install Spin Model Checker version 4.3.0 or later (http://spinroot.com/spin/Man/README.html#S1a).

3. Install eclipse plugin for TinyOS (http://tos-ide.ethz.ch/wiki/index.php)

4. Unpack the Slede archive file. Copy nesc1 file in the package into /usr/lib/ncc

  - There is already nesc1 in this directory. Back it up (i.e. nesc1-original).

  - root permission required


5. From the extracted package of Slede, copy the nesc-translator file in /usr/bin/ (root permission required).

6. From the extracted package of Slede, copy slede-plugin_1.0.0.jar in the plugins folder of eclipse.
